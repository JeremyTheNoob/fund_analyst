# 港股基准数据加载问题修复说明

## 问题描述

基金 014416（泰康研究精选股票发起A）在分析时出现以下问题：
1. 累计收益率的业绩基准曲线没有显示
2. 超额收益曲线只有一条横线（0）
3. 水下回撤的业绩比较基准曲线也没有显示

## 问题根因

### 基金基准信息
- 基金 014416 的业绩基准：恒生指数 (HSI.HI)
- 数据来源：AkShare 新浪港股指数接口 `stock_hk_index_daily_sina(symbol=symbol)`

### 错误原因

1. **港股指数代码格式错误**
   - 原始代码：`hk:HSI.HI`
   - 提取后得到：`HSI.HI`（第 653 行）
   - 新浪接口格式：`HSI`（不带 `.HI` 后缀）

2. **接口调用参数错误**
   - `base_api.py` 中的 `_ak_hk_index_daily` 函数没有正确传递 `symbol` 参数
   - 导致接口返回的不是恒生指数数据，而是其他指数数据或空数据

3. **数据加载失败**
   - 由于代码格式和参数错误，港股基准数据加载失败
   - `build_benchmark` 返回的 `benchmark.df` 为空
   - Pipeline 中的 `has_benchmark` 条件判断失败，导致 `benchmark_df` 未添加到 `chart_data`
   - 图表生成时没有基准数据，无法绘制基准曲线

## 修复方案

### 修复 1：港股指数代码格式转换

**文件**：`data_loader/equity_loader.py`（第 651-656 行）

**修改前**：
```python
elif code is not None and code.startswith("hk:"):
    # 港股指数不支持全收益计算，使用价格指数
    hk_sym = code[3:]
    df_part = load_hk_index_daily(hk_sym, start, end).rename(columns={"ret": "part_ret"})
    logger.info(f"[build_benchmark] 使用港股指数: {hk_sym}")
```

**修改后**：
```python
elif code is not None and code.startswith("hk:"):
    # 港股指数不支持全收益计算，使用价格指数
    # 转换格式: HSI.HI → HSI（新浪格式）
    hk_sym = code[3:]
    if '.' in hk_sym:
        hk_sym = hk_sym.split('.')[0]
    df_part = load_hk_index_daily(hk_sym, start, end).rename(columns={"ret": "part_ret"})
    logger.info(f"[build_benchmark] 使用港股指数: {hk_sym}")
```

### 修复 2：接口参数传递

**文件**：`data_loader/base_api.py`（第 401-413 行）

**修改前**：
```python
def _ak_hk_index_daily(symbol: str, *args, **kwargs) -> Optional[pd.DataFrame]:
    """
    港股指数日线行情（新浪）

    Args:
        symbol: 指数代码

    Returns:
        指数行情 DataFrame
    """
    return safe_api_call(
        lambda: ak.stock_hk_index_daily_sina()
    )
```

**修改后**：
```python
def _ak_hk_index_daily(symbol: str, *args, **kwargs) -> Optional[pd.DataFrame]:
    """
    港股指数日线行情（新浪）

    Args:
        symbol: 指数代码（如 "HSI"）

    Returns:
        指数行情 DataFrame
    """
    return safe_api_call(
        lambda: ak.stock_hk_index_daily_sina(symbol=symbol)
    )
```

## 相关问题

### 问题：港股指数不支持全收益计算
- **原因**：AkShare 新浪接口不提供港股指数的股息数据
- **当前处理**：港股基准使用价格指数（不含分红再投资）
- **影响**：港股基准的累计收益率可能低于实际收益率
- **改进方案**（可选）：接入 Wind 或 Bloomberg 等数据源获取港股全收益指数

### 问题：港股指数数据覆盖范围
- **来源**：AkShare 新浪接口
- **历史数据**：从 2013-08-20 开始
- **更新频率**：每个交易日
- **数据质量**：正常
