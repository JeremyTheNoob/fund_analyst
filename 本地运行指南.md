# 本地运行指南

## 1. 安装依赖

打开终端，进入项目目录：

```bash
cd /Users/liuweihua/WorkBuddy/基金穿透式分析
```

安装所需的 Python 包：

```bash
pip install -r requirements.txt
```

## 2. 运行应用

```bash
streamlit run fund_analysis.py
```

## 3. 访问应用

运行成功后，浏览器会自动打开：
```
http://localhost:8501
```

如果没有自动打开，手动在浏览器中访问上述地址。

## 4. 使用说明

1. 在左侧输入框输入基金代码（例如：110022）
2. 点击「开始分析」按钮
3. 查看基金 4 个维度的详细分析

## 常见问题

### 问题 1: 安装依赖时报错

**解决方案:**
```bash
# 升级 pip
pip install --upgrade pip

# 清理缓存后重新安装
pip cache purge
pip install -r requirements.txt
```

### 问题 2: 运行时提示 "ModuleNotFoundError"

**解决方案:**
确保在正确的目录下运行，并且已经安装了所有依赖：

```bash
# 检查目录
pwd
# 应该显示: /Users/liuweihua/WorkBuddy/基金穿透式分析

# 检查是否安装了 streamlit
pip list | grep streamlit
```

### 问题 3: 获取基金数据失败

**可能原因:**
- 基金代码不正确
- 网络连接问题（AkShare 需要访问互联网）

**解决方案:**
1. 检查基金代码是否正确（6位数字）
2. 检查网络连接
3. 尝试更换基金代码测试

### 问题 4: 图表不显示

**解决方案:**
- 刷新浏览器页面
- 清除浏览器缓存
- 尝试使用 Chrome 或 Edge 浏览器

## 停止运行

在终端中按 `Ctrl + C` 停止应用
