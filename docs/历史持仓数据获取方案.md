# 基金历史持仓数据获取方案

## 📊 问题概述

**目标**：获取基金从成立以来的持仓历史，支持以下功能：
1. 持仓变化追踪（各报告期的 Top 10 持仓）
2. 行业配置变化分析
3. 经理换人后的风格漂移检测
4. 持仓稳定性评分（基于实际留存率）

## ❌ AkShare 当前限制

| 数据需求 | AkShare 支持 | 说明 |
|---------|-------------|------|
| 最新一期 Top 10 持仓 | ✅ | `fund_portfolio_hold_em()` |
| 历史持仓序列 | ❌ | 只返回最新一期 |
| 完整报告期（Q1/Q3） | ❌ | 无中间报告期数据 |
| 季报全文（投资策略） | ❌ | 无文本数据 |

## 💡 可行解决方案

### 方案对比

| 方案 | 成本 | 数据质量 | 技术难度 | 推荐度 |
|------|------|----------|----------|--------|
| 方案一：手工导出+自动更新 | 低 | ⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| 方案二：网页爬虫 | 低 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |

## 🎯 推荐方案：方案一（手工导出+自动更新）

### 方案概述
- **历史数据**：手工从东方财富导出（一次性）
- **最新数据**：每季度自动抓取（自动化）
- **数据来源**：http://fundf10.eastmoney.com/ccmx_基金代码.html

### 数据库设计

```sql
-- 基金历史持仓表
CREATE TABLE fund_holdings_history (
    id SERIAL PRIMARY KEY,
    fund_code VARCHAR(10) NOT NULL,
    report_date DATE NOT NULL,
    stock_code VARCHAR(10) NOT NULL,
    stock_name VARCHAR(50) NOT NULL,
    holding_ratio DECIMAL(5,2) NOT NULL,  -- 持仓占比（0-100）
    rank INT NOT NULL,                    -- 持仓排名
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(fund_code, report_date, stock_code)
);

-- 基金行业配置历史表
CREATE TABLE fund_industry_history (
    id SERIAL PRIMARY KEY,
    fund_code VARCHAR(10) NOT NULL,
    report_date DATE NOT NULL,
    industry VARCHAR(50) NOT NULL,
    ratio DECIMAL(5,2) NOT NULL,  -- 行业占比
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(fund_code, report_date, industry)
);

-- 基金资产配置历史表（用于固收+）
CREATE TABLE fund_asset_history (
    id SERIAL PRIMARY KEY,
    fund_code VARCHAR(10) NOT NULL,
    report_date DATE NOT NULL,
    stock_ratio DECIMAL(5,2),      -- 股票占比
    bond_ratio DECIMAL(5,2),       -- 债券占比
    cb_ratio DECIMAL(5,2),         -- 转债占比
    cash_ratio DECIMAL(5,2),       -- 现金占比
    other_ratio DECIMAL(5,2),      -- 其他资产
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(fund_code, report_date)
);

-- 索引
CREATE INDEX idx_fund_code_date ON fund_holdings_history(fund_code, report_date);
CREATE INDEX idx_report_date ON fund_holdings_history(report_date);
```

### 数据导入流程

#### 第一步：手工导出历史数据（一次性）

**操作步骤**：
1. 访问东方财富基金页面
   - 示例：http://fundf10.eastmoney.com/ccmx_000001.html
   
2. 选择"历史持仓"标签

3. 逐期导出数据
   - 点击"下载"或"导出"按钮
   - 选择 CSV 或 Excel 格式
   - 保存到本地文件夹 `data/holdings_export/`

4. 数据格式
   ```
   基金代码,报告期,排名,股票代码,股票名称,持仓占比
   000001,2024-12-31,1,000858,五粮液,8.52
   000001,2024-12-31,2,600519,贵州茅台,7.35
   ...
   ```

**工作量**：
- 热门基金（50只）：约2-3小时
- 全部基金（约1500只）：约1-2天（可分批进行）

#### 第二步：编写导入脚本

```python
# scripts/import_holdings_history.py
import pandas as pd
from datetime import datetime
from sqlalchemy import create_engine

# 数据库连接
engine = create_engine('postgresql://user:password@localhost/fund_analytics')

def import_holdings_csv(csv_file):
    """导入持仓CSV文件到数据库"""
    df = pd.read_csv(csv_file)
    
    # 数据清洗
    df['报告期'] = pd.to_datetime(df['报告期'])
    df['持仓占比'] = df['持仓占比'].astype(float)
    
    # 写入数据库
    df.to_sql('fund_holdings_history', engine, if_exists='append', index=False)
    print(f"✅ 已导入 {len(df)} 条记录")

# 批量导入
import glob
csv_files = glob.glob('data/holdings_export/*.csv')
for csv_file in csv_files:
    import_holdings_csv(csv_file)
```

# reporter/holdings_analyzer.py（增强版）

def analyze_holding_retention(fund_code, report_dates):
    """
    计算个股留存率（需要多期数据支持）
    
    Args:
        fund_code: 基金代码
        report_dates: 报告期列表，如 ['2024-12-31', '2024-09-30', '2024-06-30']
    
    Returns:
        {
            "retention_rate": float,  # 平均留存率（0-100）
            "retention_trend": str,   # 留存率趋势（上升/下降/稳定）
            "long_term_stocks": List, # 长期持有的股票（连续3期以上）
        }
    """
    # 从数据库获取各期持仓
    holdings_history = []
    for date in report_dates:
        df = query_holdings_by_date(fund_code, date)
        holdings_history.append(df)
    
    # 计算相邻两期的留存率
    retention_rates = []
    for i in range(len(holdings_history) - 1):
        current = set(holdings_history[i]['stock_code'])
        previous = set(holdings_history[i+1]['stock_code'])
        
        if len(previous) > 0:
            retention_rate = len(current & previous) / len(previous) * 100
            retention_rates.append(retention_rate)
    
    # 平均留存率
    avg_retention = sum(retention_rates) / len(retention_rates) if retention_rates else 0
    
    # 长期持有股票（连续3期以上）
    long_term_stocks = find_long_term_stocks(holdings_history)
    
    return {
        "retention_rate": avg_retention,
        "retention_trend": get_trend(retention_rates),
        "long_term_stocks": long_term_stocks,
    }

def analyze_industry_drift(fund_code, report_dates):
    """
    分析行业配置漂移
    
    Returns:
        {
            "drift_score": float,    # 漂移评分（0-100，越高漂移越大）
            "major_changes": List,    # 重大行业变化
            "style_switch": str,     # 风格切换（如：成长→价值）
        }
    """
    # 获取各期行业配置
    industry_history = []
    for date in report_dates:
        df = query_industry_allocation_by_date(fund_code, date)
        industry_history.append(df)
    
    # 计算行业配置变化
    drifts = []
    for i in range(len(industry_history) - 1):
        current = industry_history[i]
        previous = industry_history[i+1]
        
        # 计算行业配置变化（绝对差值之和）
        drift = sum(abs(current['ratio'] - previous['ratio']))
        drifts.append(drift)
    
    # 漂移评分
    avg_drift = sum(drifts) / len(drifts) if drifts else 0
    drift_score = min(100, avg_drift / 5)  # 归一化到0-100
    
    # 重大变化（占比变化 > 10%）
    major_changes = find_major_industry_changes(industry_history)
    
    return {
        "drift_score": drift_score,
        "major_changes": major_changes,
        "style_switch": "暂需更多数据支持",
    }

```python
import jqdatasdk as jqd

# 认证
jqd.auth('your_user', 'your_password')

# 获取基金历史持仓
def get_fund_holdings(fund_code, start_date, end_date):
    """获取基金历史持仓"""
    
    # 转换基金代码（聚宽使用 0XXXXX 格式）
    jq_code = f"0{fund_code}.OF"
    
    # 获取持仓数据
    holdings = jqd.fund.get_fund_portfolio(
        code=jq_code,
        start_date=start_date,
        end_date=end_date
    )
    
    return holdings

# 使用示例
holdings = get_fund_holdings('000001', '2020-01-01', '2024-12-31')
print(holdings)
```
