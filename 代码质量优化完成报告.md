## 第三步：引入全局监控装饰器 ✅

### 3.1 audit_logger 装饰器

**功能**
1. 记录函数调用入口（函数名 + 参数）
2. 执行函数并捕获异常
3. 记录函数调用出口（执行时间 + 结果状态）
4. 异常自动记录到 ERROR 日志

**使用示例**
```python
@audit_logger
def my_function(x, y):
    return x + y
```

**日志输出**
```
2026-03-29 20:30:00 - INFO - [module:line] - ▶️ START: my_function | Args: (1, 2) | Kwargs: {}
2026-03-29 20:30:01 - INFO - [module:line] - ✅ END: my_function | Elapsed: 0.001s
```

### 3.2 工具函数

新增实用工具函数：
- `format_duration(seconds)`: 格式化时间
- `safe_divide(numerator, denominator, default)`: 安全除法
- `clip_value(value, min_val, max_val)`: 数值裁剪
