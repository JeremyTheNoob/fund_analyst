## ✅ P3: 个股基本面穿透(增强版)

### 功能描述
- 批量获取前十大重仓股的基本面数据
- 包含: 名称、价格、市值、规模标签、投资风格标签、PE、PB、ROE
- 支持市值分类(大盘>500亿/中盘100-500亿/小盘<100亿)
- 支持投资风格分类(价值/成长/均衡)

### 核心函数
```python
fetch_stock_fundamentals(stock_codes)
```

### 输出字段
```python
{
    '000001': {
        'name': '平安银行',
        'price': 10.94,
        'market_cap': 2100.5,
        'size_tag': '大盘',      # 大盘/中盘/小盘
        'style_tag': '价值',     # 价值/成长/均衡
        'pe_ratio': 5.2,
        'pb_ratio': 0.6,
        'roe': 12.5,
        'revenue_growth': 8.2,
    },
    ...
}
```

### 投资风格判定规则
- **价值**: PE<15 且 PB<2.0
- **成长**: PE>40 或 PB>8.0
- **均衡**: 其他

- 已集成重试机制(3次,延迟2秒)
- 建议增加本地缓存(TTL=1天)

## ✅ 风格标签对比(增强版)

### 功能描述
- 双维度风格对比: 市值风格 + 投资风格
- 从持仓计算平均市值风格
- 从持仓计算投资风格分布
- 从FF因子模型读取SMB(市值)和HML(价值成长)因子暴露
- 检测市值风格漂移和投资风格漂移

### 核心函数
```python
compare_style_with_ff(holdings_fundamentals, ff_results)
```

### 输出字段
```python
{
    'holding_style': {
        'size': '大盘',    # 基于持仓
        'style': '价值',   # 基于持仓
    },
    'ff_style': {
        'size': '小盘',    # 基于SMB因子
        'style': '成长',   # 基于HML因子
    },
    'is_size_consistent': False,      # 市值风格是否一致
    'is_style_consistent': True,      # 投资风格是否一致
    'note': '⚠️ 部分漂移: 投资风格一致(价值),但市值风格不同(持仓大盘 vs FF模型小盘)'
}
```

### 测试结果
```
持仓风格: 大盘价值
FF模型风格: 小盘成长
市值一致性: False (持仓大盘 vs FF小盘)
投资风格一致性: True (持仓价值 vs FF均衡)
解读: ⚠️ 部分漂移: 投资风格一致(价值),但市值风格不同(持仓大盘 vs FF模型小盘)
```

### UI展示
- 市值风格对比(三列指标卡)
- 投资风格对比(三列指标卡)
- 一致性判定图标(✅/⚠️)

## 🏗️ 技术架构

### 数据流程

```
基金持仓数据(最新+历史)
    ↓
股票代码提取
    ↓
股票→行业映射表(缓存)
    ↓
行业权重计算 ←─→ 个股基本面获取(AkShare)
    ↓                    ↓
持仓集中度计算      风格标签对比(FF因子)
    ↓                    ↓
持仓变动追踪(多期对比) ←──────┘
    ↓
持仓穿透分析结果
    ↓
UI展示(5个Tab)
```

### 缓存策略

1. **股票→行业映射表**
   - 缓存文件: `.workbuddy/cache/stock_industry_mapping.json`
   - 更新周期: 30天
   - 文件大小: 约500KB

2. **历史持仓数据**
   - 实时从AkShare获取
   - 缓存4个季度数据
   - 建议增加缓存(下次迭代)
