# 📊 基金穿透式分析工具

一键获取基金 4 维度深度分析报告：风险收益、基金经理、持仓底牌、成本规则。

## ✨ 核心功能

- 📈 **风险与收益**：夏普比率、最大回撤、年化收益、波动率
- 👤 **基金经理**：任职年限、历史回报、能力雷达图
- 📦 **持仓底牌**：前十大重仓股、集中度、持仓分布
- 💰 **成本与规则**：费率结构、分红方式、申赎状态
- 💡 **综合评分**：智能评估基金投资价值
