## 根本原因分析

### 001420 问题的根本原因

1. **基准未展示**：
   - `pipeline.py` 中的 `_run_index_pipeline` 没有将 `benchmark_df` 添加到 `chart_data`
   - 导致 `chart_gen.py` 无法获取基准数据来绘制基准曲线

2. **跟踪偏离度显示为0**：
   - `chart_gen.py` 中缺少 `tracking_error_scatter` 图表生成函数
   - main.py 中使用 `excess_return` 数据作为 fallback，但 excess_return 的数据是累计超额收益，不是逐日偏离值
   - 导致即使渲染，数据也显示错误

3. **折溢价数据均为0**：
   - `chart_gen.py` 中缺少 `premium_discount` 图表生成函数
   - 场外指数基金（非ETF）没有真实的折溢价数据，应该返回空字典让 main.py 使用 monthly_heatmap 作为 fallback

### 000047 问题的根本原因

- **字符编码问题**：字符串中使用了中文逗号 `、`（U+3001），Python 解释器无法正确识别
- 位置：`cb_report_writer.py` 第330行，`_core_conclusion_template` 函数中

### 4. 修复 001420 折溢价数据为0

**文件**: `reporter/chart_gen.py`

**修改**：添加 `_premium_discount_chart` 函数

```python
def _premium_discount_chart(report: Any) -> Dict:
    """
    折溢价图表
    如果是场外指数基金（非ETF），返回空数据，让 main.py 使用 monthly_heatmap 作为 fallback。
    """
    if not hasattr(report, 'index_metrics') or not report.index_metrics:
        return {}

    metrics = report.index_metrics
    pd_mean = getattr(metrics, 'premium_discount_mean', 0.0)
    pd_std = getattr(metrics, 'premium_discount_std', 0.0)

    # 如果折溢价数据接近 0，说明是场外基金，返回空字典
    if abs(pd_mean) < 0.001 and abs(pd_std) < 0.001:
        return {}

    # 如果有真实的折溢价数据，返回提示信息
    return {
        'type': 'message',
        'message': f'平均折溢价率: {pd_mean*100:.3f}%, 标准差: {pd_std*100:.3f}%'
    }
```

## 技术要点

### 1. 基准数据传递链路

```
index_pipeline 
  → bm_df (基准数据) 
  → FundReport.chart_data["benchmark_df"] 
  → chart_gen._cumulative_return_chart() 
  → 绘制基准曲线
```

**关键修复点**：`_run_index_pipeline` 必须将 `benchmark_df` 添加到 `chart_data`

### 2. 跟踪偏离度 vs 超额收益

- **超额收益（excess_return）**：累计超额收益，反映"1块钱投入基金比投入基准多赚了多少"
- **跟踪偏离度（tracking_error_scatter）**：逐日偏离，反映"每日基金收益率-基准收益率"

**错误原因**：之前使用 excess_return 作为 fallback，但数据类型不匹配，导致图表显示异常。

### 3. 场外基金 vs 场内ETF

- **场外基金**（如 001420）：申购赎回按净值，不存在折溢价
- **场内ETF**：二级市场交易，存在折溢价（价格 vs 净值）

**处理逻辑**：
- 如果折溢价数据接近 0，说明是场外基金，返回空字典
- main.py 会使用 monthly_heatmap 作为 fallback
- 如果有真实的折溢价数据，返回提示信息

### 4. 字符编码问题

- Python 字符串中不能直接使用中文标点符号（如 `、`）
- 应该使用英文标点或中文引号（`'` `"`）
