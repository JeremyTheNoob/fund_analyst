## 二、核心缺失功能详细说明

### 2.1 P0级（阻塞性缺失）

#### ❌ `build_benchmark_ret()` - 基准收益率构建

**位置**: 应在 `data/fetcher.py` 或 `data/processor.py`

**功能**:
- 解析招募说明书中的业绩比较基准（如"沪深300指数×70% + 中债综合×30%"）
- 支持复合基准（多个指数加权）
- 支持港股指数（`hk:`前缀识别）
- 返回带日期的基准收益率序列

**原始代码逻辑**:
```python
def build_benchmark_ret(benchmark_parsed: dict, start_date: str, end_date: str) -> pd.DataFrame:
    # 1. 遍历 components
    # 2. 对每个 component:
    #    - 如果 code.startswith('hk:'): 用 fetch_hk_index_daily
    #    - 否则: 用 fetch_index_daily
    # 3. 合并加权: Σ(weight * ret)
    # 4. 返回 DataFrame with columns: ['date', 'bm_ret']
```

**依赖数据**:
- `fetch_index_daily()` - ✅ 已实现
- `fetch_hk_index_daily()` - ✅ 已实现
- `benchmark_parsed` - 由 `fetch_basic_info()` 返回 ✅ 已实现

**影响范围**:
- `models/equity_model.py` 中的 `_build_benchmark()` 依赖此函数
- 所有权益类/混合型基金的超额收益计算
- 累计收益曲线的基准对比
