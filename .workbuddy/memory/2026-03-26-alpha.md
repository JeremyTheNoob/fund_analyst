# 2026-03-26 工作日志 - Alpha（超额收益）计算

## 任务背景
Alpha（Jensen's Alpha）是评价基金选股能力的核心指标，也是收益分解的关键组成部分。需要实现完整的Alpha计算、滚动Alpha和超额收益分解功能。

## 实现内容

### 1. 创建新文件 `models/alpha_analysis.py`

**核心函数1：`calculate_alpha()` - Alpha计算**
- **功能**：计算基金的Jensen's Alpha、超额收益、风险调整收益指标
- **输入**：
  - `fund_ret`: 基金日收益率序列
  - `benchmark_ret`: 基准日收益率序列
  - `risk_free_rate`: 无风险利率（年化，默认3%）
- **输出**：
  - `excess_return`: 超额收益（年化）
  - `alpha`: Jensen's Alpha（年化）
  - `alpha_pval`: Alpha显著性p值
  - `beta`: 系统性风险系数
  - `tracking_error`: 跟踪误差
  - `information_ratio`: 信息比率
  - `treynor_ratio`: 特雷纳比率
  - `jensen_alpha`: Jensen's Alpha（年化）
  - `r_squared`: R²拟合优度
  - `interpretation`: 文字解读

**核心函数2：`calculate_alpha_rolling()` - 滚动Alpha**
- **功能**：计算滚动窗口的Alpha，监测Alpha的稳定性
- **输入**：
  - `fund_ret`: 基金日收益率序列
  - `benchmark_ret`: 基准日收益率序列
  - `window`: 滚动窗口（默认60天）
  - `risk_free_rate`: 无风险利率（年化）
- **输出**：
  - `alpha_rolling`: 滚动Alpha序列
  - `alpha_latest`: 最新Alpha值
  - `alpha_stable`: Alpha是否稳定
  - `alpha_trend`: Alpha趋势（上升/下降/稳定）
  - `interpretation`: 文字解读

**核心函数3：`decompose_excess_return()` - 超额收益分解**
- **功能**：将超额收益分解为Alpha、Beta、大小盘因子、价值因子、残差等组成部分
- **输入**：
  - `fund_ret`: 基金日收益率序列
  - `benchmark_ret`: 基准日收益率序列
  - `ff_factors`: FF因子数据（Mkt, SMB, HML等）
- **输出**：
  - `excess_total`: 总超额收益（年化）
  - `alpha_contribution`: Alpha贡献（年化）
  - `beta_contribution`: Beta贡献（年化）
  - `size_contribution`: 大小盘因子贡献（年化）
  - `value_contribution`: 价值因子贡献（年化）
  - `residual_contribution`: 残差贡献（年化）
  - `interpretation`: 文字解读

## 功能特点

### Alpha计算特点
1. **完整的CAPM回归**：基于资本资产定价模型（CAPM）计算Jensen's Alpha
2. **显著性检验**：提供Alpha的p值，判断Alpha是否统计显著
3. **多重风险调整指标**：
   - 跟踪误差（Tracking Error）：衡量基金与基准的偏离程度
   - 信息比率（Information Ratio）：衡量超额收益的稳定性
   - 特雷纳比率（Treynor Ratio）：衡量单位系统性风险的收益
4. **拟合优度评估**：R²指标评估CAPM模型的拟合程度
5. **分层解读**：分别解读Alpha、超额收益、信息比率、特雷纳比率、R²

### 滚动Alpha特点
1. **实时监测**：滚动窗口计算Alpha，实时监测超额收益能力的变化
2. **稳定性判断**：基于Alpha序列的标准差判断Alpha是否稳定
3. **趋势检测**：线性回归检测Alpha的上升/下降/稳定趋势
4. **预警机制**：当Alpha不稳定或趋势恶化时提供预警

### 超额收益分解特点
1. **多因子分解**：将超额收益分解为Alpha、Beta、大小盘因子、价值因子、残差等
2. **贡献度分析**：识别各因子的贡献度，找出最大贡献因子
3. **可视化友好**：提供百分比形式的贡献度，便于理解和展示

## 核心公式

### Jensen's Alpha（CAPM模型）
```
R_fund - R_f = α + β × (R_market - R_f)
其中：
- R_fund: 基金收益率
- R_market: 基准收益率
- R_f: 无风险利率
- α: Jensen's Alpha（超额收益能力）
- β: 系统性风险系数
```

### 信息比率
```
信息比率 = 超额收益年化收益率 / 跟踪误差
跟踪误差 = Std(基金收益率 - 基准收益率) × √252
```

### 特雷纳比率
```
特雷纳比率 = (基金年化收益率 - 无风险利率) / Beta
```

### 超额收益分解
```
超额收益 = Alpha贡献 + Beta贡献 + 大小盘因子贡献 + 价值因子贡献 + 残差
```
