## 五、实现示例

### 5.1 可转债信息获取示例

```python
def fetch_convertible_bond_info(bond_code: str) -> dict:
    """
    获取可转债详细信息
    """
    try:
        df = ak.bond_zh_cov_info(symbol=bond_code, indicator='基本信息')
        
        if df.empty:
            return {}
        
        info = dict(zip(df.iloc[:, 0], df.iloc[:, 1]))
        
        return {
            'rating': info.get('RATING', ''),
            'expire_date': info.get('EXPIRE_DATE', ''),
            'coupon_rate': float(info.get('COUPON_IR', 0) or 0),
            'initial_price': float(info.get('INITIAL_TRANSFER_PRICE', 0) or 0),
            'stock_price': float(info.get('STOCK_PRICE', 0) or 0),
        }
    except Exception as e:
        print(f"获取可转债信息失败: {e}")
        return {}
```

### 5.2 批量获取可转债信息

```python
def fetch_bond_holdings_with_details(holdings_df: pd.DataFrame) -> pd.DataFrame:
    """
    批量获取债券持仓详细信息
    """
    # 识别可转债
    convertibles = holdings_df[
        holdings_df['债券名称'].str.contains('可转债|转债')
    ]
    
    # 批量获取
    results = []
    for idx, row in convertibles.iterrows():
        bond_code = str(row.get('债券代码', ''))
        if bond_code:
            info = fetch_convertible_bond_info(bond_code)
            results.append({
                **row.to_dict(),
                **info
            })
    
    return pd.DataFrame(results)
```

## 七、数据源获取成本

### 7.1 API调用成本
- AkShare: 免费（开源）
- 百度财经: 免费（网页爬虫）
- 新浪: 免费（公开接口）

### 7.2 存储成本
- 净值数据: 每个基金约 100KB（5年日频）
- FF因子: 全市场共享（约10MB）
- 债券三因子: 全市场共享（约5MB）
- 可转债详细信息: 每个约1KB

**总计**: 单基金约 120KB，1000基金约 120MB

### 7.3 性能优化建议
1. **公共数据缓存**: FF因子/债券因子全天缓存（ttl=86400）
2. **增量更新**: 净值数据只拉取最近30天
3. **懒加载**: 债券详细信息按需获取

## 八、风险提示

### 8.1 数据质量风险
- **AkShare接口不稳定**: 可能偶发失败，需要重试机制
- **评级数据缺失**: 部分债券可能没有评级信息
- **到期日格式不统一**: 不同数据源格式可能不同

### 8.2 合规风险
- **数据爬虫**: 百度财经爬虫可能触发反爬
- **评级数据**: 评级机构数据可能有版权问题
- **实时性**: 部分数据可能有延迟

### 8.3 技术风险
- **API变更**: AkShare接口可能随时变更
- **数据量**: 批量获取可能触发限流
- **格式变化**: 数据结构可能变化
