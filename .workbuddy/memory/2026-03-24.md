## v7.1 修复CSS TokenError

**问题**：Streamlit Cloud 部署后报错 `tokenize.TokenError`，错误指向第88行 `.tag-red { background: ... }`

**原因**：CSS 字符串直接放在模块顶层，被 Python 的源码解析器（用于 `@st.cache_data` 装饰器缓存 key 生成）误读，大括号 `{}` 和点号 `.` 被当作 Python 语法解析导致崩溃

**修复**：将 CSS 字符串移到 `render_css()` 函数内，在 `main()` 中调用，避免模块顶层解析
